# Web Technologies - Events Booking App frontend and backend

Events Booking app made up of a Laravel backend and a React frontend created with NextJS. To run the application, do the following:

- Clone the backend repository from the `ass-local` branch.
- Clone the frontend repository from the `assessment-frontend` branch.
- Open a terminal and navigate to the cloned backend
- Run the command `docker compose up` from the terminal to start up your database
- Open another terminal from the backend and run the command `php artisan serve` from the terminal to start the backend application
- Open another terminal and navigate to the cloned frontend
- Run the command `npm i` from the terminal to install the NodeJS packages
- Run the command `npm run dev` from the terminal to start the frontend app.
- Open the specified URL on your browser, usually `localhost:3000`.
- Once done, run `Ctrl + C` to exit both frontend and backend
- Shut down the database by running the command `docker compose down`
